Je veux que tu me planifie la création d'une application "Get things done" de gestion des TODO.

# Besoin
L'application tournera autour de sa restitution très lisible des todos :
 - Une treemap view qui classe les todos par priorité et effort pour les réaliser. La taille des todos représenteras l'effort à fournir pour les réaliser. La couleur représentera la priorité. Les tâches les plus urgentes et les Quick wins seront naturellement positionnées vers le haut de la vue.
 - proposera une interface de saisie et d'édition des todos :
    - Titre
    - Description
    - Effort nécessaire 1 à 5
    - Priorité 1 à 5
    - Deadline (facultatif)
 - Une tache peut être marquées done et la treemap sera mise à jour en conséquence

# Contexte technique
L'application :
 - sera développée en Angular avec Typescript et TailwindCSS.
 - proposera un mode développement local avec live reload
 - proposera une API pour de futurs clients (dashboard arduino, application mobile, ...)
